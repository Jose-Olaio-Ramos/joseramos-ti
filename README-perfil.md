## Projetos em Destaque

### 🖥️ Monitor de PDVs
> Painel web de monitoramento em tempo real dos terminais de ponto de venda (PDV) do Grupo Formosa.

- Verifica disponibilidade via **ping ICMP** e **teste de porta TCP**
- Identifica 3 estados: `ONLINE` · `PINGONLY` · `OFFLINE`
- Organizado por loja: Super, Mix, Restaurante e Farmácia
- Atualização automática a cada 60 segundos
- Stack: **PHP + Vanilla JS + CSS** (zero dependências externas)

[![Repositório](https://img.shields.io/badge/Ver_Repositório-0f1218?style=flat-square&logo=github&logoColor=white)](https://github.com/joseramos-ti/monitor-pdv)

### 📡 Monitor Busca Preços
> Painel web de monitoramento dos terminais de consulta de preços distribuídos pelo supermercado, farmácia e magazine.

- Monitoramento de **28 terminais** divididos por setor
- Ping com 3 métodos em cascata: `exec()` → `shell_exec()` → `fsockopen()`
- Interface dark com atualização assíncrona em background (sem recarregar a página)
- Contador regressivo visual e botão de verificação manual
- Stack: **PHP + Vanilla JS + CSS** (zero dependências externas)

[![Repositório](https://img.shields.io/badge/Ver_Repositório-0f1218?style=flat-square&logo=github&logoColor=white)](https://github.com/joseramos-ti/monitor-busca-precos)
